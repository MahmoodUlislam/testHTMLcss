Design 103

This course is designed for specialized training in the design of web applications. Students will learn to design and develop web applications using current web development tools and techniques. Students will also learn to design web applications that are accessible to all users.

Course Outcomes

Upon successful completion of this course, students will be able to:

  • Design and develop web applications using current web development tools and techniques.
  • Design web applications that are accessible to all users.
  • Design web applications that are secure and maintainable.
  • Design web applications that are responsive to different screen sizes.
  • Design web applications that are compatible with different browsers.
  • Design web applications that are compatible with different operating systems.
  • Design web applications that are compatible with different devices.
  • Design web applications that are compatible with different web servers.
  • Design web applications that are compatible with different web browsers.
